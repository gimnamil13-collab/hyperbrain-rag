# Zettelkasten (제텔카스텐) 요약

독일 사회학자 니클라스 루만(Niklas Luhmann)이 사용한 카드 노트 시스템.

## 기본 아이디어

- 하나의 노트 = **하나의 아이디어**
- 노트끼리 **링크**로 연결
- 태그보다 연결이 중요

## Atomic Note (원자 노트)

한 노트에는 한 가지 생각만 담는다.
예: "RAG는 검색 + 생성의 조합이다" → 별도 노트

## Permanent Note (영구 노트)

남의 글을 베끼지 않고 **내 말로** 다시 쓴다.
이게 Second Brain의 핵심 자산이 된다.

## RAG 활용 예시

질문: "루만의 노트 시스템에서 가장 중요한 건 뭐야?"
→ Zettelkasten 노트에서 "링크"와 "내 말로 쓰기" 관련 구절을 찾아 답변

## 실천 팁

- 매일 노트 1개만 써도 1년이면 365개
- 제목은 검색하기 쉽게 문장형으로
- PDF 하이라이트 → Markdown으로 옮기면 RAG 품질이 좋아진다
